# Demo Dataset

Create **20 candidates** for one role: **AI Engineer**.

## Groups

| Group | Count | Profile | Purpose |
|-------|-------|---------|---------|
| A | 5 | MIT/Google/OpenAI credentials | Obvious strong candidates |
| B | 10 | Normal resumes | Average candidates |
| C | 5 | Unknown college, strong GitHub/projects | **Hidden talent showcase** |

## Target Demo Moment

- ATS ranks hidden talent candidate at **#15**
- DELULU ranks same candidate at **#2**

## Deliverable

The most important artifact is `candidate_dataset_v1.json` — without a carefully designed dataset, DELULU cannot demonstrate its value.

Use `shared/candidate_schema.json` as the template for each entry.
